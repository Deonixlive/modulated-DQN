# modulated-DQN
Begleitcode zu meiner Maturarbeit, welches sich um eine modifizierte Version von DQN beschäftigt.

## Anhängigkeiten
Es werden die folgenden Module und Libraries benötigt:
- Python 3 >= 3.8
- tensorflow==2.9.2
- tensorflow-probability==0.17.0
- Ray[rllib] >= 2.0.0
- gym[atari]==0.23.1
- lz4==4.0.2
